## ⚙️ Getting Started Locally

1. **Clone the repo:**
   ```bash
   git clone https://github.com/mell0wx/brain-dump-site.git
   cd brain-dump-site
   ```

2. **Install dependencies:**
   ```bash
   bundle install
   ```

3. **Serve Locally:**
   ```bash
   bundle exec jekyll build
   ```

   4. **Visit**
   http://localhost:4000/
